Snow¶

SIA snow model¶

actions.snow.SIA_snow.Ce(windExposition)¶

wind exposition coefficient for snow load. according SIA 261 2014 (5.2.4).

actions.snow.SIA_snow.PsSIA160_1970(h)¶

characteristic value of snow load. according SIA 160 1970 (23.1). h: altitude (m).

actions.snow.SIA_snow.Sk(h0)¶

characteristic value of snow load. according SIA 261 2014 (5.2.6). h0: reference altitude (annex D)

actions.snow.SIA_snow.qkPont(h0, windExposition)¶

characteristic value of snow load on bridges (horizontal deck). according SIA 261 2014 (5.2.2).

actions.snow.SIA_snow.qkRoof(mui, Ct, h0, windExposition)¶

characteristic value of snow load on roofs. according SIA 261 2014 (5.2.2). mui: coefficient that represents shape of the roof (see 5.2.3). Ct: thermal coefficient (sell 5.2.5)
